# 📂 Pocketly — Your Very Cool Budgeting Partner

## Problem Statement

Since this is a very personal project, I decided to do something that I would actually use. I realized that i'm a very "money-calculative" person. Not that I know how to manage money well but I like to break it down. I had tried the 30%, 20%, 10% rule and others but i wanted something custom. I used to do my calculations on paper and my very tired XR's calculator and that was a hassle. Then it clicked, Why can't i make something that does these calculations for me easily and i can actually track where my money goes ? That's how pocketly was born.

---

Pocketly is a mobile-responsive budgeting application designed to help users manage personal finance allocations with intention and composure. By matching a clean frontend with an integrated browser cache persistence strategy and secure third-party account session loops, Pocketly delivers robust data boundaries between profiles without running complex server clusters.

---

## 🚀 Key Functional Modules

- ** Dual-Mode Sign In/Sign Up Portal**: Seamless authentication featuring secure Email/Password pipelines alongside interactive Google Pop-up handshakes.
- ** Onboarding Budget Wizard**: A two-step setup loop tracking macro capital pool distributions (Income, Savings %, Spending %) and micro envelope category ceilings.
- **Dynamic Core Sanctuary Dashboard**: Responsive multi-tier metric visualization tracking real-time unallocated cash balances and single-click bank credential clipboard access.
- **Transaction Log Stream**: Live localized expense ledger updates with instant conditional warning alerts when an assigned category ceiling approaches limits (>= 85%).
- **Audited Data Intelligence Reports**: Layout print-optimized analytics modules compiling capital velocity breakdowns, daily burn rates, and peak allocation footprints.

---

## 🛠️ Core Technology Stack

- **Runtime Environment**: Node.js & Vite
- **Frontend Framework**: React 18 (Hooks, Context Provider Architecture)
- **Navigation Matrix**: React Router DOM v6 (Private Route Security Guards)
- **Styling & Responsive Layout Engine**: Tailwind CSS (Fluid Mobile-first Breakpoints Matrix)
- **Backend & Token Authorization Pipeline**: Firebase v10 SDK (Auth Engine Core)
- **Icons Management Toolkit**: React Icons (`fa`, `fa6`, `ci`, `bi`)
